# ArduinoWithGolang_ServoMotorContral

-----

Using Golang and Gobot(https://gobot.io/) package.


min max angle = 0 <= angle <= 180

- press arrow left  = angle - 1
- press arrow right = angle + 1
- press arrow up    = random angle
